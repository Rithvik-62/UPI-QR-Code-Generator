# UPI QR Code Generator using Python

A simple Python project that generates UPI payment QR codes for PhonePe, Paytm, and Google Pay using a user-provided UPI ID.

## 🚀 Features

* Generate UPI QR codes <br>
* Save QR codes as image files<br>
* Supports multiple payment apps<br>

## 🛠 Technologies Used
* Python<br>
* qrcode library<br>
* Pillow (PIL)<br>
## ▶️ Run the Project

```bash id="grnyd2"
pip install qrcode[pil]
python upi_qr_generator.py
```

Learned QR code generation, Python libraries, and UPI payment URL handling.
